# 快速启动指南

## 🚀 快速启动

### 方式一：使用启动脚本（推荐）

**启动前端服务：**
```bash
./start-frontend.sh
```

**启动后端服务（新终端窗口）：**
```bash
./start-backend.sh
```

### 方式二：手动启动

**启动前端服务：**
```bash
cd frontend
npm run dev
```

**启动后端服务（新终端窗口）：**
```bash
cd backend
source venv/bin/activate
python main.py
```

## 📍 访问地址

服务启动成功后，访问以下地址：

- **前端应用**: http://localhost:3003 ⚠️ 注意是 3003
- **后端 API**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs

## 功能说明

### 1. 上传项目
- 在首页上传 ZIP 格式的项目文件
- 系统将自动解析代码结构
- 生成项目架构图

### 2. 生成群聊剧本
- 在项目页面输入业务场景（如"用户登录"、"注册流程"）
- 系统将生成拟人化的群聊对话
- 展示各个组件之间的交互过程

### 3. 查看架构图
- 左侧显示 Mermaid 格式的架构图
- 可视化展示项目结构

## 停止服务

### 停止后端
在运行后端的终端窗口按 `Ctrl+C`

### 停止前端
在运行前端的终端窗口按 `Ctrl+C`

## 常见问题

### 后端启动失败
1. 确认已激活虚拟环境：`source venv/bin/activate`
2. 检查依赖是否安装：`pip list | grep fastapi`
3. 查看错误日志

### 端口被占用
如果 8000 或 3000 端口被占用，可以修改配置文件：
- 后端端口：修改 `backend/.env` 中的 `PORT`
- 前端端口：修改 `frontend/vite.config.ts` 中的 `server.port`
